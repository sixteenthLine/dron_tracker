# Drone Object Tracking Prototype

## Overview
A prototype for real-time object tracking from drone video streams using lightweight computer vision methods.

## Motivation
The goal was to build a tracking pipeline suitable for UAV applications where inference speed and responsiveness are important.

## Features
- Motion-based tracking
- GOTURN tracker experiment
- RTMP stream processing
- Python/OpenCV implementation

## My contribution
- Implemented tracking pipeline
- Tested different tracking approaches
- Explored real-time constraints for drone-based object following

Python, OpenCV, computer vision, object tracking
